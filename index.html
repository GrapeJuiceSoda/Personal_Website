What is this website?

This website is comprised of boxes. Each box on the website has a outerbox, bar box, and a body box. This allows the website to be seperated into sections and adding on to the website is as easy as creating a new box. The blog webpages are seperated into 3 sections. The links, the blogs, and the images. Links are on the far left, blogs go in the middle, and images are on the far right. Each of these sections are know as the outer boxes. The smaller boxes inside the outerbox all have the same margins and sizes automatically making the site look clean and sleek without using bloated 3rd party website generators.
